# BetterBags_EllesmereTheme

EllesmereUI-styled theme for [BetterBags](https://github.com/Cidan/BetterBags) windows.

Restyles the BetterBags backpack and bank to match the [EllesmereUI](https://www.curseforge.com/wow/addons/ellesmereui) design language: dark panels with 1px borders, a flat header band, EllesmereUI-configured fonts, and tabs that highlight with your live EllesmereUI accent color.

## Installation

1. Go to the [Releases](../../releases/latest) page and download the latest `.zip`
2. Extract the **BetterBags_EllesmereTheme** folder into your addons directory:
   ```
   World of Warcraft\_retail_\Interface\AddOns\
   ```
3. Log in to WoW or type `/reload` in-game

## Usage

Open the BetterBags options (`/bb` or left-click the bag button → Options), go to **Themes**, and select **EllesmereUI**. The theme applies instantly.

## Requirements

- World of Warcraft Retail (Midnight / 12.x+)
- [BetterBags](https://www.curseforge.com/wow/addons/better-bags) (required dependency)
- [EllesmereUI](https://www.curseforge.com/wow/addons/ellesmereui) — the theme only shows as available when EllesmereUI is loaded; it reads the accent color, fonts, and close-button icon from it

## Notes

- Item slot buttons keep BetterBags' default styling (including Masque support).
- The theme is created with decoration frames only — no BetterBags or EllesmereUI files are modified.
